# knowledge/raw/

Immutable source-of-truth inputs (articles, PDFs, transcripts).

After compile, material typically moves here from `knowledge/inbox/`. Treat as read-only for agents.

> **Private by design.** In the public source repository, `raw/` is
> gitignored — no personal source data ships. Knowledge pages may cite
> paths like `knowledge/raw/articles/Thread by @karpathy.md` for
> provenance tracking; these files exist only in the installed vault
> (`$LLM_WIKI_ROOT`), not in the public clone. The path references are
> intentional and correct; the files themselves are private.
